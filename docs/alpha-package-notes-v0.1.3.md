# Desktop Organizer v0.1.3-alpha Package Notes

## Package Type

This package is a Windows x64 self-contained single-file build for direct internal testing.

## Package Contents

- `DesktopOrganizer.exe`

Use this build for real-machine validation of:

- startup behavior;
- group creation and deletion;
- rename flow;
- drag and drop;
- persistence after restart.
