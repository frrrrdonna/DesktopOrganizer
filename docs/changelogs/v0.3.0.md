# v0.3.0 - 2026-07-03

## 版本概述

这一版本将桌面篮子的交互从固定边缘停靠升级为可自由拖拽、可自动吸附边缘、可独立滚动的浮动布局，并同步按新规则整理版本文档。

### Basket 模型新增字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `Basket.X` | `double` | 篮子当前的水平位置。 |
| `Basket.Y` | `double` | 篮子当前的垂直位置。 |
| `Basket.SnapEdge` | `BasketDockEdge?` | 当前吸附的边缘；`null` 表示自由布局。 |

### BasketViewModel 新增能力

| 成员 | 说明 |
|------|------|
| `X`、`Y`、`SnapEdge` | 与模型双向同步的可观察属性。 |
| `ApplyDragDelta(dx, dy)` | 处理拖拽增量，并在拖动时解除原有吸附。 |
| `SnapToNearestEdge()` | 在接近屏幕边缘时自动吸附。 |
| `ClampX/ClampY()` | 限制篮子不被拖出可见区域。 |

## 已知问题

- 刷新桌面后，用户手动调整的篮子位置会被重新计算。
- 篮子位置目前仍未持久化到磁盘。
- 吸附边缘时没有过渡动画。
- 多显示器场景下的初始布局仍偏向主屏逻辑。
- 标题栏拖拽和折叠按钮在极快点击时可能出现轻微交互冲突。
