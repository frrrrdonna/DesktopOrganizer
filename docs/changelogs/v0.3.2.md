# v0.3.2 — 2026-07-03

## 版本概述

这个小版本统一了篮子的初始大小（利用 v0.3.0 新增的滚动能力），为标题栏增加了双击折叠/恢复交互，并添加了中文语言支持。

## 新增

| 文件 | 说明 |
|------|------|
| `src/DesktopOrganizer.App/Services/LocalizationService.cs` | 多语言字符串服务，支持 en-US 和 zh-CN，通过静态属性和 INotifyPropertyChanged 实例支持 XAML 绑定。 |
| `DesktopHostSettings.Language` | 新增语言偏好字段，默认 `"en-US"`，持久化在设置中。 |
| `SettingsViewModel.SelectedLanguage` / `SupportedLanguages` | 设置界面中的语言选择器绑定。 |
| `SettingsWindow.xaml` → 语言 ComboBox | 设置窗口中新增语言下拉选择框。 |

### 篮子交互增强

| 变更 | 说明 |
|------|------|
| 双击标题栏折叠/恢复 | `BasketView.xaml` 的 Thumb 标题栏新增 `MouseDoubleClick` 事件 → `ToggleCollapseCommand`。 |
| 统一篮子初始高度 | `BasketViewModel.PanelHeight` 属性：展开时固定 400px，折叠时 auto；`BasketView.xaml` 外层 Border 绑定 `Height="{Binding PanelHeight}"`。 |

- 多显示器场景下的初始布局仍偏向主屏。
