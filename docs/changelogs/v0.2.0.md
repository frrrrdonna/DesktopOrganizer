# codex v0.2.0 - 2026-07-03

## Version overview

This version introduces the first desktop-layer integration skeleton for Desktop Organizer, including desktop icon visibility control, desktop scanning, auto-generated baskets, a desktop host window, and a settings window.

## Known Issues

- WorkerW attachment is best-effort and does not yet guarantee exact Fences-style shell integration behavior.
- Desktop host settings are not persisted yet.
- Desktop baskets are generated from scan results but are not yet writable back into a physical desktop layout.
- Existing organizer windows and workspace persistence remain in the codebase but are no longer the primary startup path.
