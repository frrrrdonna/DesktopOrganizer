# v0.3.4 - 2026-07-03

## 版本概述

这个 PATCH 版本修复了语言切换后宿主 UI 不即时刷新的问题，并为 `v0.3.3` 引入的宿主持久化链路补充了自动化测试覆盖。

## 新增

| 文件 | 说明 |
|------|------|
| `src/DesktopOrganizer.Core/Models/CoordinateClamper.cs` | 提取坐标 clamp 为可测试的公共静态方法。 |
| `tests/DesktopOrganizer.Core.Tests/CoordinateClamperTests.cs` | `ClampX` / `ClampY` 边界测试，共 7 个用例。 |
| `tests/DesktopOrganizer.Core.Tests/DesktopClassifierTests.cs` | 稳定 `Key` 分配、分类路由、空篮子过滤测试，共 5 个用例。 |
| `tests/DesktopOrganizer.Core.Tests/JsonDesktopHostStateStoreTests.cs` | 默认加载与 `Save/Load` round-trip 测试，共 2 个用例。 |
| `docs/github-release-v0.3.4.md` | `v0.3.4` 的 GitHub Release 中文正文源文件。 |

## 变更

| 文件 | 变更内容 |
|------|----------|
| `src/DesktopOrganizer.App/Services/TrayIcon.cs` | 新增 `ClearAllItems()`、`UpdateTooltip()` 方法；构造函数接收 `tooltip` 参数替代硬编码提示文本。 |
| `src/DesktopOrganizer.App/App.xaml.cs` | 抽取 `BuildTrayMenu()`；语言切换时重建托盘菜单并同步更新 tooltip。 |
| `src/DesktopOrganizer.App/ViewModels/DesktopHostViewModel.cs` | 订阅 `LocalizationService.Instance.PropertyChanged`；新增 `RefreshStatusText()` 用于在语言切换时刷新状态文本；改为调用 `CoordinateClamper`。 |
| `tests/DesktopOrganizer.Core.Tests/DesktopOrganizer.Core.Tests.csproj` | 新增对 `DesktopOrganizer.Infrastructure` 的项目引用，用于持久化链路测试。 |
| `src/DesktopOrganizer.App/DesktopOrganizer.App.csproj` | 版本号提升到 `0.3.4`。 |
| `docs/bug-tracker.md` | 补充“语言切换后宿主界面未即时刷新”的缺陷记录。 |

## 移除

无。

## 已知问题

- 多显示器场景下的布局恢复仍使用虚拟屏幕坐标。
- 篮子身份仍限于分类器定义的默认 `key`，用户暂无法自定义。
- 桌面扫描结果（实际文件列表）仍不持久化。
