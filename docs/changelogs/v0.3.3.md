# v0.3.3 - 2026-07-03

## Version Overview

This release consolidates the local `v0.3.1`, `v0.3.2`, and `v0.3.3` work into a single GitHub release.

It closes the tray-exit desktop icon restore bug, adds localized desktop-host UI, improves basket interaction, and introduces persisted desktop-host state so layout and settings survive restarts.

## Known Issues

- Language changes made in the settings window do not fully re-render the host window until the application is restarted.
- Basket placement still uses virtual screen coordinates and does not yet have multi-monitor-aware restore rules.
- Basket identity is limited to classifier-defined default keys; users cannot define custom persistent basket identities yet.
- Scanned desktop item membership is still generated at runtime and is not persisted to disk.
