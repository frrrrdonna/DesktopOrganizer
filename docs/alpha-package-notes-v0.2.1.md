# Desktop Organizer v0.2.1-alpha Package Notes

## Package Type

This package is a Windows x64 self-contained single-file build for direct internal testing.

## Package Focus

This alpha build focuses on the desktop-layer prototype:

- transparent desktop overlay;
- desktop icon hide and restore flow;
- floating baskets generated from desktop content;
- tray-based exit path;
- abnormal-exit recovery snapshot.

## Test Goal

Use this build to validate:

- startup and transparent desktop host behavior;
- desktop icon recovery safety;
- basket rendering and item launching;
- tray menu and settings entry;
- exit and restart behavior.

## Delivery Form

Primary output:

- publish folder

Convenience output:

- zip archive of the publish folder
