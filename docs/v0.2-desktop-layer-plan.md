# v0.2 Desktop Layer Plan

## Goal

Move Desktop Organizer from an ordinary organizer window toward a desktop-layer component that behaves closer to Fences.

## Ordered Plan

1. Hide and restore original desktop icons safely.
2. Launch a desktop host surface instead of the normal organizer window.
3. Scan current desktop content from user and common desktop locations.
4. Auto-classify scanned content into generated baskets.
5. Render baskets docked to screen edges with collapse support.
6. Add a settings window for desktop-host behavior.
7. Later phase: replace logical mapping with optional physical desktop-file relocation.

## What v0.2.0 Implements

- desktop icon hide/restore service;
- desktop content scanner;
- default basket classifier;
- desktop host window;
- edge-based basket layout;
- settings window;
- attempt to attach the WPF host window to the desktop layer.

## What v0.2.0 Does Not Implement Yet

- physical movement of existing desktop files;
- full WorkerW/Explorer parity with Fences behavior;
- persisted basket layout and settings;
- shell replacement;
- multi-monitor basket placement rules.
