# Desktop Organizer v0.1.3-alpha

## Alpha 版本说明

这是 `Desktop Organizer` 的首个可实际测试的 Alpha 版本。

## 包含内容

- Windows x64 自包含单文件测试包
- 支持创建分组、重命名、折叠与展开
- 支持拖拽文件、文件夹、快捷方式到分组
- 支持条目打开、删除确认与工作区持久化

## 已验证

- 程序可正常启动
- `Release` 构建通过
- 单文件 `publish` 成功
- 打包 `zip` 已生成并验证可用

## 已知限制

- 当前仍为 Alpha 版本
- 还不是安装器形态
- 仍使用默认程序图标
- 组布局仍是原型方案，不是最终桌面布局系统
